# Storybook Doc Blocks Disponíveis

## Blocos que podemos usar no Introduction.mdx:

1. **Meta** - Controla a posição na sidebar (já usado)
2. **Canvas** - Wrapper em torno de uma Story com toolbar interativa e source code
3. **Story** - Renderiza stories do CSF dentro do MDX com todas as annotations
4. **Source** - Renderiza snippets de código
5. **ColorPalette / ColorItem** - Documenta cores (já usado em Colors.mdx)
6. **IconGallery** - Grid de ícones
7. **Typeset** - Documenta fontes
8. **Title** - Heading primário
9. **Subtitle** - Heading secundário
10. **Description** - Descrição do componente via JSDoc
11. **Primary** - Renderiza a story primária
12. **Controls** - Tabela dinâmica de args
13. **ArgTypes** - Tabela estática de arg types
14. **Stories** - Coleção completa de stories
15. **TableOfContents** - TOC fixo na lateral direita
16. **Unstyled** - Remove estilos padrão do Storybook (útil para layouts custom)
17. **Markdown** - Importa markdown puro

## Blocos mais interessantes para a Introduction:
- **Unstyled** - Para criar layouts totalmente custom sem interferência do Storybook
- **Story** + **Canvas** - Para embedar previews interativas de componentes diretamente na intro
- **Source** - Para mostrar snippets de código de instalação/uso
- **IconGallery** - Se tivermos ícones
- JSX inline custom (como já usado em Typography.mdx e Spacing.mdx)
