# Spacing & Radius

Virtu UI uses a deliberate spacing and border-radius system built on CSS custom properties. Every component in the library uses these tokens exclusively — no hardcoded pixel values. This guarantees visual consistency across your product and makes global layout adjustments trivial.

All tokens are prefixed with `--virtu-`.

---

## Spacing Scale

The spacing scale follows a base-4 progression, providing a predictable rhythm for margins, paddings, and gaps. Use these tokens whenever you need to position or separate elements.

<div style={{ display: "flex", flexDirection: "column", gap: "12px", margin: "24px 0" }}>
  {[
    { token: "--virtu-spacing-xs", value: "4px",  px: 4  },
    { token: "--virtu-spacing-sm", value: "8px",  px: 8  },
    { token: "--virtu-spacing-md", value: "16px", px: 16 },
    { token: "--virtu-spacing-lg", value: "24px", px: 24 },
    { token: "--virtu-spacing-xl", value: "32px", px: 32 },
  ].map(({ token, value, px }) => (
    <div key={token} style={{ display: "grid", gridTemplateColumns: "220px 60px 1fr", alignItems: "center", gap: "16px", padding: "12px 16px", borderRadius: "10px", background: "rgba(255, 255, 255, 0.06)", border: "1px solid rgba(255, 255, 255, 0.08)" }}>
      <code style={{ fontSize: "12px", color: "rgba(255, 255, 255, 0.6)" }}>{token}</code>
      <span style={{ fontSize: "12px", color: "rgba(255, 255, 255, 0.4)", fontVariantNumeric: "tabular-nums" }}>{value}</span>
      <div style={{ display: "flex", alignItems: "center", gap: "10px" }}>
        <div style={{ width: `${px * 4}px`, height: "20px", background: "#FF7F72", borderRadius: "3px", flexShrink: 0 }} />
        <span style={{ fontSize: "11px", color: "rgba(255, 255, 255, 0.3)" }}>{px * 4}px visual bar (4x)</span>
      </div>
    </div>
  ))}
</div>

## Border Radius

Border-radius tokens define the rounding applied to interactive elements such as buttons, inputs, cards, and the `Box` component. The `--virtu-radius-lg` token (15px) is the signature radius used by the glassmorphism `Box` container.

<div style={{ display: "flex", flexDirection: "column", gap: "12px", margin: "24px 0" }}>
  {[
    { token: "--virtu-radius-sm", value: "6px", radius: "6px" },
    { token: "--virtu-radius-md", value: "10px", radius: "10px" },
    { token: "--virtu-radius-lg", value: "15px", radius: "15px" },
  ].map(({ token, value, radius }) => (
    <div key={token} style={{ display: "grid", gridTemplateColumns: "200px 60px 1fr", alignItems: "center", gap: "16px", padding: "12px 16px", borderRadius: "10px", background: "rgba(255, 255, 255, 0.06)", border: "1px solid rgba(255, 255, 255, 0.08)" }}>
      <code style={{ fontSize: "12px", color: "rgba(255, 255, 255, 0.6)" }}>{token}</code>
      <span style={{ fontSize: "12px", color: "rgba(255, 255, 255, 0.4)", fontVariantNumeric: "tabular-nums" }}>{value}</span>
      <div style={{ display: "flex", alignItems: "center", gap: "12px" }}>
        <div style={{ width: "80px", height: "32px", background: "rgba(255, 255, 255, 0.06)", border: "1px solid rgba(255, 255, 255, 0.12)", borderRadius: radius }} />
        <div style={{ width: "32px", height: "32px", background: "#FF7F72", borderRadius: radius }} />
      </div>
    </div>
  ))}
</div>

## Usage

```css
/* Glass card */
.card {
  padding: var(--virtu-spacing-md);
  border-radius: var(--virtu-radius-lg);
  background: var(--virtu-glass-gradient);
  backdrop-filter: blur(var(--virtu-glass-blur));
  border: var(--virtu-glass-border);
}

/* Form field gap */
.form {
  display: flex;
  flex-direction: column;
  gap: var(--virtu-spacing-sm);
}

/* Button padding */
.button {
  padding: var(--virtu-spacing-sm) var(--virtu-spacing-md);
  border-radius: var(--virtu-radius-md);
}
```
