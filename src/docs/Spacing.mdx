# Spacing & Radius

Boulder UI uses a deliberate spacing and border-radius system built on CSS custom properties. Every component in the library uses these tokens exclusively — no hardcoded pixel values. This guarantees visual consistency across your product and makes global layout adjustments trivial.

All tokens are prefixed with `--boulder-`.

---

## Spacing Scale

The spacing scale follows a base-4 progression, providing a predictable rhythm for margins, paddings, and gaps. Use these tokens whenever you need to position or separate elements.

<div style={{ display: "flex", flexDirection: "column", gap: "12px", margin: "24px 0" }}>
  {[
    { token: "--boulder-spacing-xs", value: "4px",  px: 4  },
    { token: "--boulder-spacing-sm", value: "8px",  px: 8  },
    { token: "--boulder-spacing-md", value: "16px", px: 16 },
    { token: "--boulder-spacing-lg", value: "24px", px: 24 },
  ].map(({ token, value, px }) => (
    <div key={token} style={{ display: "grid", gridTemplateColumns: "220px 60px 1fr", alignItems: "center", gap: "16px", padding: "12px 16px", borderRadius: "6px", background: "#f9f9f9", border: "1px solid #ececec" }}>
      <code style={{ fontSize: "12px", color: "#555" }}>{token}</code>
      <span style={{ fontSize: "12px", color: "#999", fontVariantNumeric: "tabular-nums" }}>{value}</span>
      <div style={{ display: "flex", alignItems: "center", gap: "10px" }}>
        <div style={{ width: `${px * 4}px`, height: "20px", background: "#8a7650", borderRadius: "3px", flexShrink: 0 }} />
        <span style={{ fontSize: "11px", color: "#aaa" }}>{px * 4}px visual bar (4×)</span>
      </div>
    </div>
  ))}
</div>

---

## Border Radius

Border-radius tokens define the rounding applied to interactive elements such as buttons, inputs, and cards. Consistent rounding contributes to a cohesive and polished visual identity.

<div style={{ display: "flex", flexDirection: "column", gap: "12px", margin: "24px 0" }}>
  {[
    { token: "--boulder-radius-sm", value: "4px", radius: "4px" },
    { token: "--boulder-radius-md", value: "8px", radius: "8px" },
  ].map(({ token, value, radius }) => (
    <div key={token} style={{ display: "grid", gridTemplateColumns: "200px 60px 1fr", alignItems: "center", gap: "16px", padding: "12px 16px", borderRadius: "6px", background: "#f9f9f9", border: "1px solid #ececec" }}>
      <code style={{ fontSize: "12px", color: "#555" }}>{token}</code>
      <span style={{ fontSize: "12px", color: "#999", fontVariantNumeric: "tabular-nums" }}>{value}</span>
      <div style={{ display: "flex", alignItems: "center", gap: "12px" }}>
        <div style={{ width: "80px", height: "32px", background: "#8a7650", borderRadius: radius }} />
        <div style={{ width: "32px", height: "32px", background: "#dbcea5", border: "2px solid #8a7650", borderRadius: radius }} />
      </div>
    </div>
  ))}
</div>

---

## Usage

```css
/* Card */
.card {
  padding: var(--boulder-spacing-md);
  border-radius: var(--boulder-radius-md);
  border: 1px solid var(--boulder-color-border);
}

/* Form field gap */
.form {
  display: flex;
  flex-direction: column;
  gap: var(--boulder-spacing-sm);
}

/* Button padding */
.button {
  padding: var(--boulder-spacing-sm) var(--boulder-spacing-md);
  border-radius: var(--boulder-radius-sm);
}
```
