# Colors

The Virtu UI color palette is designed around a **dark, glassmorphism-first** aesthetic. Colors are defined as CSS custom properties so that consumers can override any value to match their brand without touching component source code.

All color tokens are prefixed with `--virtu-color-`.

---

These are the foundational dark tones from which all semantic roles are derived. They reflect the dark, immersive identity of Virtu UI.

## Glass Effect Tokens

Virtu UI provides dedicated tokens for the glassmorphism effect used by the `Box` component and other surfaces.

| Token | Value | Description |
|---|---|---|
| `--virtu-glass-gradient` | `linear-gradient(135deg, rgba(172,172,172,0.19), rgba(70,70,70,0.19))` | Diagonal gradient from bottom-left to top-right |
| `--virtu-glass-blur` | `24px` | Backdrop blur intensity |
| `--virtu-glass-border` | `1px solid rgba(255,255,255,0.08)` | Subtle light border |
| `--virtu-glass-shadow` | `0 8px 32px rgba(0,0,0,0.25)` | Soft drop shadow |

---

## Customization

Override any token once in your application's `:root` to retheme the entire library:

```css
:root {
  /* Swap the primary accent color */
  --virtu-color-primary: #6366f1;
  --virtu-color-primary-hover: #4f46e5;

  /* Adjust the glass intensity */
  --virtu-glass-blur: 32px;
}
```

No component-level changes are needed. All components that reference these tokens will reflect the new values automatically.
