    A **lightweight React component library** built around the **glassmorphism** design language. Translucent surfaces, subtle gradients, and backdrop blur effects create modern, elegant interfaces inspired by iOS aesthetics.

    Virtu UI is **font-agnostic** — it ships with `system-ui` as the default font stack, but consumers can freely override the typography tokens to use any font of their choice.

    ---

    ### Glassmorphism

    Components feature **translucent backgrounds with backdrop blur**, creating a layered depth effect that feels modern and immersive.

    ### Accessibility

    All components follow **accessibility best practices** and include proper semantic structure and ARIA attributes when necessary.

    ### Consistency

    A set of **design tokens** ensures consistent styling across applications and components, with full support for dark interfaces.

    ---

    ## Available Components
    - Box
    - Button
    - Input
    - Textarea
    - Checkbox
    - Switch
    - Label
    - FormField
    - ErrorMessage
    - Accordion
    - Badge

    ---

    ## Quick Start

    ```bash
    npm install virtu-ui
    ```

    ```tsx
    import "virtu-ui/styles";
    import { Box, Button } from "virtu-ui";
    ```
