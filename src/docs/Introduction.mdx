    A **lightweight React component library** designed to build **consistent, accessible and composable interfaces**.

    ---

    ### Simplicity

    Components are **minimal and composable**, allowing developers to build complex interfaces with simple building blocks.

    ### Accessibility

    All components follow **accessibility best practices** and include proper semantic structure and ARIA attributes when necessary.

    ### Consistency

    A set of **design tokens** ensures consistent styling across applications and components.

    ---

    ## Available Components
    - Button
    - Input
    - Textarea
    - Checkbox
    - Switch
    - Label
    - FormField
    - ErrorMessage

    ---

    ## Quick Start

    ```bash
    npm install boulder-ui
    ```
