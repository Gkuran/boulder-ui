# Boulder UI

Em progresso

**Boulder UI** is a lightweight React component library designed to help developers build consistent and accessible interfaces.

The library focuses on **simplicity, accessibility, and design token driven styling**, making it easy to build scalable user interfaces while maintaining visual consistency.

---

## Philosophy

Boulder UI is built around three core principles:

### Simplicity

Components are intentionally minimal and flexible, allowing developers to extend them easily without unnecessary abstraction.

### Accessibility

All components are built with accessibility in mind, supporting keyboard navigation, screen readers, and semantic HTML.

### Design Tokens

Boulder UI relies on **CSS design tokens** to ensure consistency across components, including colors, spacing, typography, and radius values.
