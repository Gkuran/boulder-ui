# Typography

Boulder UI provides a flexible and scalable typography system through CSS custom properties. This system ensures visual consistency while allowing consumers to easily integrate their preferred fonts and adjust text properties to match their brand guidelines.

All typography tokens are prefixed with `--boulder-`.

## Font Family (Agnostic Approach)

Boulder UI is **font-agnostic**. We do not enforce a specific font family. By default, the library uses `system-ui, sans-serif`, which leverages the native fonts of the user's operating system, providing fast loading times and a familiar feel.

```css
:root {
  --boulder-font-family: system-ui, sans-serif;
}
```

### Customization

You can easily override the font family to integrate your own brand font. Simply declare your desired font in your application's CSS:

```css
:root {
  --boulder-font-family:
    "Inter", sans-serif; /* Example: Using the Inter font */
}
```

This approach ensures that all components using `--boulder-font-family` will automatically adopt your chosen font, maintaining consistency across your application.

## Usage

To apply these typography tokens in your CSS, reference them as CSS custom properties:

```css
.my-heading {
  font-family: var(--boulder-font-family);
  font-size: var(--boulder-font-size-lg);
  font-weight: var(--boulder-font-weight-bold);
  line-height: var(--boulder-line-height-sm);
}
```

By consistently using these tokens, you ensure a cohesive and maintainable typographic scale across your product.
