# Typography

Virtu UI provides a flexible and scalable typography system through CSS custom properties. This system ensures visual consistency while allowing consumers to easily integrate their preferred fonts and adjust text properties to match their brand guidelines.

All typography tokens are prefixed with `--virtu-`.

---

## Font Family — Agnostic Approach

Virtu UI is **font-agnostic**. The library does not enforce or bundle any specific typeface. By default, `--virtu-font-family` resolves to `system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif`, which leverages the native font stack of the user's operating system — providing zero-latency rendering and a familiar feel without any additional loading cost.

```css
:root {
  --virtu-font-family: system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI",
    Roboto, sans-serif;
}
```

### Bringing your own font

To integrate your brand's typeface, override the token once in your application's global CSS. Every Virtu UI component that renders text will automatically adopt the new value.

```css
/* Example: Google Fonts — Inter */
@import url("https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap");

:root {
  --virtu-font-family: "Inter", sans-serif;
}
```

No component-level changes are required. The single token override propagates through the entire system.

## Usage

```css
.heading {
  font-family: var(--virtu-font-family);
  font-size: var(--virtu-font-size-xl);
  font-weight: var(--virtu-font-weight-bold);
  line-height: var(--virtu-line-height-sm);
  color: var(--virtu-color-text);
}

.body {
  font-family: var(--virtu-font-family);
  font-size: var(--virtu-font-size-md);
  font-weight: var(--virtu-font-weight-regular);
  line-height: var(--virtu-line-height-md);
  color: var(--virtu-color-text-secondary);
}
```
